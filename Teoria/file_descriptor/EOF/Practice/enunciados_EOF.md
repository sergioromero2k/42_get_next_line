#### Problema 1 – Contar caracteres

Escribe un programa que lea caracteres desde el teclado hasta que se detecte EOF y luego muestre cuántos caracteres se ingresaron.

#### Problema 2 – Contar líneas

Escribe un programa que lea texto desde un archivo o desde teclado y cuente **el número de líneas** hasta EOF.
```
Pista: una línea termina con \n.
```
#### Problema 3 – Contar palabras

Escribe un programa que lea texto desde la entrada estándar y cuente **el número de palabras** hasta EOF.
```
Considera que las palabras están separadas por espacios, tabulaciones o saltos de línea.
```
#### Problema 4 – Copiar texto

Escribe un programa que copie todo lo que el usuario escriba hasta EOF y lo muestre inmediatamente en la pantalla (como un “eco”).

#### Problema 5 – Contar ocurrencias de un carácter

Escribe un programa que lea texto hasta EOF y cuente cuántas veces aparece un carácter específico ingresado por el usuario.
